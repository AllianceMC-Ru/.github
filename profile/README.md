## 🎮 О нас

**AllianceMC-Ru** — закрытое сообщество разработчиков и администраторов Minecraft-серверов.  
Мы занимаемся разработкой плагинов, конфигураций и инфраструктуры для русскоязычного Minecraft-сообщества.
