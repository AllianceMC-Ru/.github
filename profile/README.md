## 🎮 О нас

**AllianceMC-Ru** — закрытое сообщество разработчиков и администраторов Minecraft-серверов.  
Мы занимаемся разработкой плагинов, конфигураций и инфраструктуры для русскоязычного Minecraft-сообщества.

## 🚀 Быстрый старт для участников

1. Откройте нужный репозиторий организации.
2. Внимательно прочитайте `README.md` и инструкции по установке.
3. Перед началом работы синхронизируйте локальную ветку с `main`.
4. Делайте небольшие, логически завершённые коммиты с понятными сообщениями.
5. Для крупных изменений создавайте Pull Request с описанием мотивации и результата.
